# High-Agent: Production-Grade Dynamic AI Agent System

## Overview
Highest-caliber AI agent with piecewise regime switching, agent-skills, Taste Skill integration, and Textual TUI. Built for "throw any task" reliability on Arch Linux / Hyprland with local LLMs.

## Architecture (Floor Plan)
- **Dynamic Regimes**: Piecewise switching (simple → complex tasks)
- **Skills Layer**: agent-skills + taste-skill + custom piecewise
- **Orchestration**: CrewAI / LangGraph
- **UI**: Textual TUI (expandable to GUI)
- **Memory**: Holographic / Honcho
- **Hardware**: Optimized for RTX 5080
